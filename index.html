Rocoin

This project was developed in the summer of 2024.

The purpose of this game and this blog is to learn and practice programming.

In Rocoin, the objective is doge rocks and collect coins.

This game was developed in Python with the Pygame module.

A total of 157 lines of code.

There is the game itself, the main menu, and the death screen.

The main menu includes a play and exit button.

The death screen displays how many coins the player has collected.
